# vendor-assessment-qcc

> **供应商评估Skill（企查查MCP增强版）**

基于 [Panaversity Supply Chain Plugin](https://github.com/panaversity/agentfactory-business-plugins) 的 vendor-assessment Skill 增强开发。

## 增强内容

### 原版功能
- ✅ Kraljic 矩阵供应商分类
- ✅ 六维度评估框架
- ✅ 标准输出格式

### QCC MCP 增强
- 🆕 9维度评估模型（基础6维 + 供应链特有3维）
- 🆕 18类风险实时扫描
- 🆕 实体锚定（统一社会信用代码验证）
- 🆕 产能资质、组织稳定性、业务健康度评估

## 文件说明

- `SKILL.original.md` - 原作者Skill定义
- `SKILL.qcc-enhanced.md` - QCC MCP增强版
- `SKILL.md` - 当前使用版本
- `qcc-mcp-integration/` - QCC MCP连接器代码
